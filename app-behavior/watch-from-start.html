Introduction

Watch from Start is an easy access point for viewers to start a VOD asset. This way, viewers can resume the content and watch a particular video. This is for Roku and SmartTV devices since beginning a program over using the remote control is challenging sometimes.

Notes:
  • Applicable to Details I only
  • This only applies to Roku, LG, Samsung Tizen, AndroidTV, and FireTV. It does not apply to web, mobile, and AppleTV, as restarting videos on those devices is easy.

Notice how you see a restart button on details page I for the following content types:

  • Movie
  • Episodes

You can watch content from the beginning when you select the restart button. You will see the button only when you have watched any portion of a given video. The button won’t be shown when a video hasn’t been started.
