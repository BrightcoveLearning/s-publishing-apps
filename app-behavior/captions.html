Overview

In VideoCloud, you have the capability to add labels to each track made available to apps through the Playback API. These labels are essential for allowing users to select the desired text track for playback.

Label Property Usage

When adding text tracks, ensure that the label property is used. This property serves as a user-friendly identifier for text tracks.

Handling Empty Labels

If the label property is empty, the platform displays the locally available translation for the language code associated with the text track.

Managing Multiple Tracks without Labels

In cases where multiple text tracks exist for the same language (e.g., en-US, en-GB) and none of them have labels, the following format is used for the text track title:

  1. Language Variation: Include the language variation in the title (e.g., "English (United States)", "English (British)").
  2. Text Track Type: Also, include the type of text track in the title (e.g., "Captions (en)", "Subtitle (en-GB)").

Applicability

This labeling system is applicable to Player A and Player E.

By following these guidelines for labeling text tracks, you can provide a seamless user experience for selecting and displaying text tracks in Brightcove Beacon apps players.
