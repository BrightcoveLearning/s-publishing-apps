Overview

In your favorite OTT app, you can return and continue watching a series based on what episode you last completed, so you can continue enjoying the show in the expected order. This also applies to movies. In case you have partially watched more than one series, the continue watching row should show you the next episode in the series when you complete watching an episode.

View All Grid

The playlist thumbnails display a playback progress bar for movies and episodes, indicating the watched portion. This applies to:

  • Continue Watching Playlist
  • View All Page of the Continue Watching Playlist
  • Details I Episodes Page
  • Player E Episodes Related List
  • Player E Movies Related List
  • Assets types:
    • Movies
    • Episodes.
    • Note: Once you have created your continue watching playlist in Beacon Classic, you can see it in Beacon Studio, but you can only edit it in Beacon Classic.

Playback progress bar

  • Once you finish the episode, it no longer appears in continue watching
  • The playback progress bar does not appear if there has been 0 watch history on that episode
  • For episodes completed in the account, the progress bar shows full completion of playback and remains with that status unless the user begins watching the episode over again
  • Preview players (Auto Play, Live TV preview and EPG Preview) should not report bookmarks to Beacon Classic
  • It is shown when navigating the view all page for the continue watching playlist
  • Applies to both portrait and landscape thumbnail images
  • It is updated per platform each time the page is refreshed
  • It is sync across platforms only on opening/launching the app.

Set Up for Beacon Classic

To set up your account for continue watching, follow this steps:

  1. Login to Beacon Classic
  2. In the top navigation, click Playlists
  3. Click Add new playlist
  4. To create the continue watching playlist, you have to:
    1. Name your playlist
    2. Check the Special playlist box
    3. In Special Type, select Continue Watching (100)

    To see learn more about playlist, see Creating a playlist.

  5. In Assets filters, configure the buttons for the assets you want to turn on and off.
  6. Click Save changes in Playlist

    7. Now you can view the playlist in your Beacon apps.
