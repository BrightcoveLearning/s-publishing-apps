Introduction

In your Beacon apps' movies and episodic content you can set cue points that cause Beacon to display a button for the viewer to skip a video's introduction and/or credits. You have the option to set four cue points in a video that control when the appropriate button appears:

  • skip_intro_start
  • skip_intro_end
  • skip_credits_start
  • skip_credits_end

The following list the behavior of the cue points and buttons:

  • If you set a skip_intro_end, but not skip_intro_start, the beginning of video assumed to be the starting point.
  • If you set a skip_credit_start, but not skip_credit_end, the end of video assumed to be the ending point.
  • If an end cue point comes before a corresponding start cue point, nothing happens.
  • If there is a duplication of any cue points, the second one is ignored.
  • There is no validation of logical placement of cue points at this time.
  • Pre-roll and post-roll ads play irregardless of cue point placement.
  • If the skip credits button is clicked, and there is an ad inserted before the end of the credits, all ads play.
  • By default, the skip buttons are displayed for 10 seconds before they disappear. This default can be changed by contacting Brightcove's OTT Delivery Team.

Adding cue points

  1. Log in to Beacon Studio.
  2. Go to the Media module.
  3. Click the video on which you wish to implement skip functionality. This displays the video's details.
