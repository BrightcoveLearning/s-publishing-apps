Introduction

In your Beacon apps' movies and episodic content you can set cue points that cause Beacon to display a button for the viewer to skip a video's introduction and/or credits. You have the option to set the following four cue points in a video that control when the appropriate button appears:

  • skip_intro_start
  • skip_intro_end
  • skip_credits_start
  • skip_credits_end

The following lists the behavior of the cue points and buttons:

  • If you set a skip_intro_end, but not a skip_intro_start, the beginning of the video is assumed to be the starting point.
  • If you set a skip_credit_start, but not a skip_credit_end, the end of video is assumed to be the ending point.
  • If an end cue point comes before a corresponding start cue point, nothing happens.
  • If there is a duplication of any cue points, the second one in the timeline is ignored.
  • Pre-roll and post-roll ads play irregardless of cue point placement.
  • If the skip credits button is clicked, and there is an ad inserted before the end of the credits, all ads play.
  • By default, the skip buttons are displayed for 10 seconds before they disappear, or at the end of the intro/credits stop cue points, whichever is sooner. This default of 10 seconds can be changed by contacting Brightcove's OTT Delivery Team.
  • There is no validation of logical placement of cue points at this time.

Once the cue points have been created, in the Beacon app when a movie or episode is playing a small Skip Intro or Skip Credits button is displayed at the desired times.

Contact the Brightcove OTT Delivery Team to customize the Skip Intro or Skip Credits button.

Adding cue points

The following short video (1:23) demonstrates the process of adding cue points, and a demonstration of use the buttons implemented by the cue points.

Following the video are step-by-step instructions detailing the same process shown in the video.

 
  1. Log in to Beacon Studio.
  2. Go to the Media module.
  3. Click the video on which you wish to implement skip functionality. This displays the video's details.
  4. To add the cue points, do the following:
    1. Use the scrubber to position the playhead to the desired point in the video.
    2. Click Add cue point.
    3. Be sure the Type is code.
    4. Type or paste in the desired cue point name. Your options are:
      • skip_intro_start
      • skip_intro_end
      • skip_credits_start
      • skip_credits_end
      Be careful here as there is no validation at this time checking for typos or illogically placed cue points.
    5. Click Save.
  5. Repeat the process for all needed cue points.
