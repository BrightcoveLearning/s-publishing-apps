Introduction

To make your channels and Beacon assets available in Roku search you need to complete the following process:

  • Generate a data report using Studio's External Search Feeds functionality. Specifically, you will create a template that uses template that that uses the Roku Search Feed 2.0 (JSON) format type. Then create and run a job that generates data for you video assets. Detailed steps are included in the linked document.
  • Setup the Roku Search Feed 2.0 in Roku's Developer Dashboard.
