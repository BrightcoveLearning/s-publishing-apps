Introduction

This document shows you how to install a test build so you don't have to go through the app submission process before you can test your app.

A high-level view of the process is:

  1. Create a development account on the LG developer site.
  2. Connect to your LG WebOS device through the Android Debug Bridge (adb).
  3. Install your Brightcove {{ site.product_short }} App.
  4. Run your Brightcove {{ site.product_short }} App.

Creating a developer account

To use the Developer Mode app, you will need a development account on the LG developer portal site.

  1. In a broswer, navigate to the LG webOS TV Developer site.

  2. In the top right, click Sign In.
  3. Either sign in or select Create Account. When creating a new account, a Select Country dialog will appear.
  4. After you have created an account, install the Developer mode app on your TV.

Configuring your account

With the Developer Mode app installed on your TV, you will sign in and customize some settings.

  1. Launch the Developer mode app on your TV. Enter your credentials from your account created on the LG Developer site.

  2. Turn on the following properties:
    • Dev Mode Status - to turn this option on, you will need to reboot the TV
    • Key Server
    • Clear Cache Data - this is optional
  3. Leave the TV on because you will need the IP Adress and Passphrase to connect to the TV.

Installing the webOS IDE

Now, you are ready to install the webOS IDE on your computer.

Running your Brightcove {{ site.product_short }} App

Sideloaded apps appear in both the Recent row and in the My Library row in the Apps section. You can also find your app in the Settings menu.

  1. From the Amazon Fire TV main screen, select Settings > Applications > Manage Installed Applications.
  2. Select the Brightcove {{ site.product_short }} app.
  3. Select Launch application.

Uninstalling your Brightcove {{ site.product_short }} App

To uninstall your app from the Fire TV device follow these steps:

  1. From Fire TV's main screen, select Settings > Applications > Manage Installed Applications.
  2. Select your Brightcove {{ site.product_short }} app.
  3. Select Uninstall > Uninstall
